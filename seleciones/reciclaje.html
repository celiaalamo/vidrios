Procesos de Fabricación
Actualmente, los principales procesos de fabricación de envases de vidrio son:
el soplado-soplado (botellas de boca estrecha)
el prensado-soplado (recipientes de boca ancha)
RECICLAJE:
Es un proceso circular; una vez que el envase ha sido utilizado y depositado para su reciclaje, comienza el proceso de reciclaje:
Primero se separa el vidrio de cuerpos extraños, se tritura y se limpia. Después es llevado a las fábricas de vidrio donde se utiliza como materia prima para la nueva elaboración del vidrio. El producto resultante de las plantas de tratamiento se denomina "calcín” en la industria vidriera, donde se volverá a utilizar igual como si fuera materia prima nueva.
El calcín se mezcla con arena, sosa, caliza y otros componentes y se funde a 1.500 grados centígrados. Después, el vidrio es homogeneizado hasta obtener una masa en estado líquido, de la que se extrae la llamada la gota de vidrio. Por último esta gota se lleva al molde, que dará forma al nuevo envase. Estos envases tienen las mismas características que los originales.
El vidrio recuperado se denomina vidrio reciclado.
